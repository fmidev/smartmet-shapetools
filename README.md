# smartmet-shapetools

Part of [SmartMet Server](https://github.com/fmidev/smartmet-library-spine). See the [SmartMet Server documentation](https://github.com/fmidev/smartmet-library-spine) for an overview of the ecosystem.

## Overview

Command-line tools for working with ESRI shapefiles in the SmartMet ecosystem. Used to prepare and manipulate geographic boundary data for use in weather map products.

## Features

- ESRI shapefile inspection and manipulation
- Coordinate transformation and filtering
- Shapefile format conversion utilities
